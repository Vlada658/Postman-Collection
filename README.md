# Postman-Collection
**Описание задания**  
Разработана коллекция в Postman для тестирования API интернет-магазина на основе Swagger-документации.

**Что было сделано:**  
- Изучена Swagger-документация API  
- Создана коллекция DemoShopping в Postman  
- Настроены запросы для модулей:
  - Products  
  - Cart  
- Реализована авторизация для работы с корзиной  
- Созданы переменные окружения (base URL, key, product ID)  
- Добавлены автотесты для проверки:
  - статус-кодов  
  - структуры и типов данных в ответе  
  - времени ответа
