# Postman-Collection
**Описание задания**  
Разработана коллекция в Postman для тестирования API интернет-магазина на основе Swagger-документации.

**Что было сделано:**  
- Изучена Swagger-документация API  
- На основе документации вручную созданы запросы в Postman  
- Реализована структура коллекции (Products, Cart)  
- Настроена авторизация для работы с корзиной  
- Созданы переменные окружения (base URL, key, product ID )  
- Добавлены автотесты для проверки:
  - статус-кодов  
  - структуры и типов данных  
  - времени ответа
